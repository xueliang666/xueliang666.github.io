GeosparkViz 可视化

GeoSparkViz 可视化

上一节我们使用GeoSpark SQL简单进行了空间的一些操作，本节我们继续利用GeoSpark SQL以及GeoSparkViz将我们的结果进行渲染展示。下面是我们今天要用到的新的6个SQL函数：

  1. CAST ( expression AS type ):这是PostGresql原生提供的函数，用于字段类型转换
  2. ST_Point (X:decimal, Y:decimal, UUID1, UUID2, ...)：从给定的X和Y生成一个点要素。
  3. ST_Pixelize (A:geometry, ResolutionX:int, ResolutionY:int, Boundary:geometry)：将给定的geometry转为图片中的像素。
  4. ST_Colorize (weight:Double, maxWeight:Double, mandatory color: string (Optional))：给定一个像素的权重，返回相应的颜色。权重可以是任意含义的数值，比如温度和湿度等观测值。
  5. ST_Render (A:pixel, B:color)：给定像素和颜色，生成Java中的BufferedImage对象，代表了一副图片。
  6. ST_Envelope_Aggr (A:geometryColumn)：返回A的最小外接矩形
