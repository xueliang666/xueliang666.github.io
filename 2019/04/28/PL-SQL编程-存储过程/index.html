PL/SQL编程-存储过程

存储过程实际上就是一种命名的PL/SQL程序块。

创建存储过程

创建存储过程需要使用procedure关键字。
创建存储过程不需要使用DECLARE关键字，转而使用CREATE/REPLACE关键字。
